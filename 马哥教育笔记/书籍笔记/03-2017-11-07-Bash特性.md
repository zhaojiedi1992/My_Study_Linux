# LINUX 上bash的基础特性
## 命令历史
环境变量
* HISTSIZE:命令历史的条数
* HISTFILE:默认为`~/.bash_history`
* HISTFILESIZE:HISTFILE文件记录历史的条数

history 常用命令
* -d: 删除指定的命令
* -c：清空命令
* -a:手工追加当前会话的命令历史到历史文件中去
* n :显示最近的n条命令

调用历史命令
* !#:重复执行第#条命令
* !!:重复执行上一条命令
* !str:执行指定str开头的命令（最后一个）
* !?:获得最后执行的状态码

控制命令历史的记录方式

主要和HISTCONTROL这个环境变量有关（`/etc/profile`）
* ignoredups:忽略重复
* ignorespace:忽略空白开头
* ignoreboth:上面2个都启用
