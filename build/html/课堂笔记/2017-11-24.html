1.11.2. 本节内容¶

[]里面的测试都需要加个""。
=~使用扩展的表达式

双中括号里面才支持扩展正则表达式
==支持不支持正则， 支持glob

[ -v  var1 ]： 测试变量是否没有赋值

()是一个子进程执行
{}是一个函数执行

-r:实际的上是否有读取权限


read 没有设置，存放在REPLY中

echo $-: himBH

read -s：隐藏输入
read -n:只能输入指定字符数量

-samefile : 相同文件， 也就是硬链接的文件


$-:
    h： hash
    i:  交互模式
    m： job监控
    B: 大括号扩展
    H: 历史可用
