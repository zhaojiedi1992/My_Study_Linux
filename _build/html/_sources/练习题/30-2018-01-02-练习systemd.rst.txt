3、修改默认的启动内核为新编译内核

.. code-block:: text

	1. 查看新内核在/boot/grub2/gurb.cfg里面的menuententry。
	2. 修改/etc/default/grub中GRUB_DEFAULT=1
	3. grub2-mkconfig
	4. 重启即可


4、启动时临时禁用SELinux

.. code-block:: text

	selinux=0

5、启动时进入emergency模式

.. code-block:: text

	systemd.unit=emergency.target

6、卸载编译安装的新内核

.. code-block:: bash

	1. 删除/boot目录对应文件
	2. 删除/lib/modules对应文件
	3. grub2-mkconfig 
	4. 重启
